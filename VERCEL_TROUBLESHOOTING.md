# 🚨 خطوات إصلاح Vercel

## الخطوة 1️⃣: إعادة النشر اليدوية

1. **اذهب إلى Vercel Dashboard**
   - https://vercel.com/dashboard

2. **اختر مشروعك: 3alosh**

3. **اضغط على Deployments**

4. **اختر آخر deployment واضغط: Redeploy**
   ```
   Click: ... (ثلاث نقاط)
   Select: Redeploy
   ```

5. **انتظر حتى ينتهي ثم اختبر**

---

## الخطوة 2️⃣: تحقق من متغيرات البيئة

اذهب إلى: **Settings → Environment Variables**

تأكد من وجود **جميع** هذه المتغيرات:

```
✓ TELEGRAM_BOT_TOKEN = (من @BotFather - طويل جداً)
✓ VITE_TELEGRAM_BOT_USERNAME = (اسم البوت فقط بدون @)
✓ MONGODB_URI = (رابط كامل من MongoDB Atlas)
✓ MONGODB_DB = tliker
✓ ADMIN_PASSWORD = (كلمة قوية)
✓ ALLOW_WEB_ADMIN_LOGIN = true
```

**إذا كانت ناقصة أو خاطئة:**
- أضفها من جديد
- ثم اضغط: **Redeploy** من جديد

---

## الخطوة 3️⃣: افحص Logs للبحث عن أخطاء

1. اذهب إلى **Deployments**
2. اختر التحديث الأخير
3. اضغط على **Logs**
4. ابحث عن أخطاء (كلمات باللون الأحمر)

**الأخطاء الشائعة:**
```
❌ "Cannot find module" → مشكلة في الاستيراد
❌ "MONGODB_URI is undefined" → متغير البيئة ناقص
❌ "Cannot connect to database" → تحقق من MongoDB URI
❌ "Timeout" → المشروع يستغرق وقتاً طويلاً للبناء
```

---

## الخطوة 4️⃣: اختبر البوت

بعد الـ Redeploy الناجح:

1. **افتح البوت على Telegram**
   ```
   @VITE_TELEGRAM_BOT_USERNAME
   ```

2. **انقر على أي زر وتحقق من:**
   - ✓ تحميل الصفحة الرئيسية
   - ✓ الإنتقال إلى صفحات أخرى
   - ✓ ظهور زر "السحب" في رقم التنقل السفلي

3. **انقر على زر السحب (⬆️)**
   - يجب أن ترى صفحة السحب مع:
     - عرض الرصيد الحالي
     - نموذج إدخال المبلغ
     - اختيار طريقة الدفع
     - زر "طلب السحب"

---

## الخطوة 5️⃣ (إضافية): إعادة تثبيت المشروع

إذا لم ينجح شيء من الأعلى:

1. **اذهب إلى Settings → General**

2. **اضغط على: Delete Project**

3. **ثم اضغط على: + Add New → Project**

4. **اختر نفس Repository من GitHub مرة أخرى**

5. **أضف المتغيرات من جديد**

6. **اضغط Deploy**

---

## 📋 قائمة تفقد سريعة

- [ ] تم عمل Redeploy على Vercel؟
- [ ] تم التحقق من جميع متغيرات البيئة؟
- [ ] المتغيرات القيم صحيحة (بدون مسافات إضافية)؟
- [ ] الـ MONGODB_URI صحيح ومحدث؟
- [ ] افتح الـ Logs وتحقق من الأخطاء؟
- [ ] انتظرت 2-3 دقائق بعد Redeploy؟
- [ ] جرب البوت من جديد بعد التحديث؟

---

## 🆘 إذا استمرت المشكلة

1. **خذ لقطة شاشة من Logs** (الأخطاء الحمراء)
2. **التقط الرابط الكامل للمشروع من Vercel**
3. **أرسلها لي مع وصف المشكلة**

---

**ملاحظة مهمة:**
- تحديثات GitHub → Vercel تحتاج 2-3 دقائق
- إذا عملت تغييراً محلياً، يجب عمل:
  ```bash
  git push origin main
  ```
  ثم Redeploy على Vercel
